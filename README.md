# Exercícios - Java Programming MOOC

Este repositório contém minhas soluções para os exercícios do curso **Java Programming**, oferecido pela Universidade de Helsinki.

O objetivo deste repositório é registrar minha evolução no aprendizado de Java e praticar conceitos fundamentais da linguagem.

## Conteúdos estudados

Durante o curso são abordados temas como:

- Fundamentos da linguagem Java
- Estruturas condicionais
- Estruturas de repetição
- Métodos
- Programação Orientada a Objetos (POO)
- Coleções e estruturas de dados básicas

## Curso

Curso: Java Programming MOOC  
Instituição: University of Helsinki  
Link: https://java-programming.mooc.fi

## Observação

Alguns exercícios podem estar incompletos, pois o repositório é atualizado conforme avanço no curso ou posso ter pulado pois não vi dificuldades nele.
